Conditional Statements

If you wanted to execute code in conditional manner, then we need conditional statements

Types of Conditional statements

  • 1. SWITCH Statement
  • 2. IF-ELSE Statement

IF ELSE

if (condition) { // STATEMENT } else if (condition) { // STATEMENT } else if (condition) { // STATEMENT } else { // STATEMENT }
Example
var switchStatus = false; if (switchStatus == true) { console.log("Switch is on") } else { console.log("Switch is off") }

Loops

If you wanted to run group of statements or a set of functions for N times continuosly then we should be using loops

Types of Loops

  • FOR Loop
  • While Loop
  • Do While

FOR Loop Syntax

for (START; END; STEPS) { // Statements }

Example

// Print from 1 to 10
for (var x = 1; x <= 10; x=x+1) { console.log(x) }

WHILE Loop Syntax

while(condition) { }

Example

// Print from 1 to 10
var x = 1; while(x <= 10) { console.log(x) x = x + 1; }
